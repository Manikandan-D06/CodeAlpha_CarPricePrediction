# 🚗 Car Price Prediction

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://codealphacarpriceprediction-bva29s7qzx5jdghx53hvl3.streamlit.app/)

Predicting car resale prices with machine learning — built using Python, scikit‑learn, and Streamlit.

---

## 📖 About the Project
Car Price Prediction is a machine learning project that estimates the resale value of cars based on key features such as present price, year of manufacture, kilometers driven, fuel type, transmission, and seller type.  
The pipeline includes data preprocessing, model training (Linear Regression and Random Forest), evaluation, and deployment through a Streamlit app.  
It demonstrates end‑to‑end ML workflow design, from exploratory data analysis to interactive prediction, making it a strong portfolio project for showcasing applied data science skills.

---

## ✨ Features
- Data preprocessing and feature engineering  
- Model training with Linear Regression & Random Forest  
- Model evaluation and comparison  
- Interactive Streamlit app for predictions  
- Clean project structure with modular code  

---

## 📊 Results
- Achieved competitive accuracy on test data  
- Random Forest performed better than Linear Regression  
- Visualizations included for feature importance and prediction demo
